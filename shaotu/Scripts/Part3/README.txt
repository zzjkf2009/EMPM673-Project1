This folder contains 4 files

1. CroppedBuoy.mat is the parameter of Cropped Buoy

2. estimate.m is function that compute mean and covariance

3. detectBuoy.m is function that uses gaussain mixture model with 2 
components. 

4. TestFrame.m is script to test function 'estimate.m' and 
'detectBuoy.m'. !!! Before run this script, please change link to your 
own test frame folder !!!
