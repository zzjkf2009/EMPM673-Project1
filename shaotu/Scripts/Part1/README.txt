Part 1 folder contains 4 files

1. estimate.m is function to compute mean and covariance

2. detectBuoy.m is function to detect buoy in Frames

3. TestFrame.m is script to test function 'estimate.m' and 
'detectBuoy.m'. Before run this file, please change to your own frame 
directory in line 6, imread('.........').

4. CroppedBuoy.mat is saved parameter for detectBuoy.m function.
